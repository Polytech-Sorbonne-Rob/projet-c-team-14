◆ moveCameraMan()

void moveCameraMan ( FILE *  arduino,
int  key,
int *  q0,
int *  q1 
)

Permet au pan-tilt d'être mis en mouvement manuellement. Elle modifie directement les angles dans le main. Il suffit d'appuyer sur les touches Z Q S D pour mettre en mouvement le pan-tilt.

Paramètres
[in]arduinoAdresse des ports de l'arduino.
[in]keyTouche sur laquelle l'utilisateur appuie.
[out]q0Angle Pan du pan-tilt.
[out]q1Angle Tilt du pan-tilt.
